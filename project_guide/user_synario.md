### **시나리오 1 : 일부개정 조문 Diff 분석 및 리스크 등급 산정**

- ID : SC-001
- 상황 :
    
    안전보건팀 담당자가 **산업안전보건법 일부개정 입법예고**를 확인하였다.
    
    해당 개정이 우리 조직의 점검 주기 및 법적 책임에 영향을 주는지 빠르게 판단해야 한다.
    
    수작업으로 조문 비교하지 않고,
    
    **변경 내용 + 리스크 등급 + 조직 영향 여부**를 5분 이내에 파악하고자 한다.
    
- 목표 :
    - 조문 단위 변경 탐지
    - 변경 유형 자동 분류
    - 법적 책임 관점 리스크 등급 산정
    - 조직 적용 여부 판단
- 사전 조건 :
    - 법령 구조 DB 최신 상태
    - 대상 법령: 산업안전보건법
    - 개정 유형: 일부개정
    - 조직 프로파일:
        - 제조 사업장 운영
        - 위험물 취급 있음
        - 정기 점검 연 1회 수행 중
- 상세 흐름
    
    
    | 단계 | 사용자 행동 | 시스템 동작 | 비고 |
    | --- | --- | --- | --- |
    | 1 | 입법예고 원문 업로드 | Orchestrator 유형 분류 | 일부개정 판별 |
    | 2 | - | Change Analyst (DIFF Mode) 호출 | 조문 단위 diff |
    | 3 | - | Risk Analyst 호출 | 의무 강도 분석 |
    | 4 | - | Control Auditor 호출 | 조직 적용 여부 판단 |
    | 5 | - | Action Strategist 호출 | 실행 과제 도출 |
    | 6 | - | Final Synthesis 생성 | 종합 리포트 |
- 입력 예시
    
    ```
    산업안전보건법 일부개정안
    제23조 제2항
    "매년 1회 이상 점검" → "6개월마다 1회 이상 점검"
    ```
    
- 기대 출력
    
    ```
    [변경 요약]
    - 제23조 제2항 점검 주기 변경
    - 연 1회 → 반기 1회 (빈도 2배 증가)
    
    [리스크 분석]
    - 의무 강도 증가
    - 처벌 규정 유지
    - 관리 부담 증가
    리스크 등급: Medium
    
    [조직 영향]
    - 현재 점검 주기: 연 1회
    - 법령 요구사항 미충족
    
    [Action]
    1. 점검 매뉴얼 개정 (High)
    2. 내부 점검 프로세스 수정 (High)
    3. 교육 자료 업데이트 (Medium)
    ```
    
- 성공 기준
    - 변경 조문 100% 정확 탐지
    - 리스크 등급 수동 평가 대비 90% 이상 일치
    - 전체 처리 시간 5분 이내

### **시나리오 2 : 신설 조문 구조 분석 및 적용 대상 판별**

- ID : SC-002
- 상황 :
    
    환경 관련 법령에 **신설 조문이 추가되는 입법예고**가 발표되었다.
    
    신설 의무가 우리 조직에 적용되는지 빠르게 확인해야 한다.
    
- 목표 :
    - 신설 조문 구조 분석
    - 적용 대상 조건 추출
    - 조직 프로파일과 매칭
- 사전 조건 :
    - 개정 유형: 신설
    - 조직 프로파일:
        - 화학물질 저장시설 보유
        - 대기배출시설 보유 없음
- 상세 흐름
    
    
    | 단계 | 사용자 행동 | 시스템 동작 | 비고 |
    | --- | --- | --- | --- |
    | 1 | 입법예고 입력 | Orchestrator 유형 분류 | 신설 판별 |
    | 2 | - | Change Analyst (STRUCTURE Mode) | 구조 분석 |
    | 3 | - | Risk Analyst | 의무 강도 평가 |
    | 4 | - | Control Auditor | 적용 여부 판단 |
    | 5 | - | Action Strategist | 대응 전략 수립 |
- 입력 예시
    
    ```
    제45조의2 신설
    화학물질 저장시설을 운영하는 사업자는 분기 1회 이상 안전점검을 실시하여야 한다.
    ```
    
- 기대 출력
    
    ```
    [신설 요약]
    - 화학물질 저장시설 대상 분기 점검 의무 신설
    
    [적용 여부]
    - 조직 프로파일상 저장시설 보유
    - 적용 대상 해당
    
    [리스크]
    - 의무 불이행 시 과태료 500만원
    리스크 등급: High
    
    [Action]
    1. 분기 점검 체계 수립
    2. 담당자 지정
    3. 예산 반영 검토
    ```
    
- 성공 기준
    - 적용 대상 조건 정확 추출
    - 조직 적용 여부 판단 정확도 90% 이상
    - False Positive 5% 이하

### **시나리오 3 : 삭제 조문 영향 분석**

- ID : SC-003
- 상황 :
    
    기존 의무 조항이 삭제되는 일부개정안이 발표되었다.
    관리 부담 감소 여부와 내부 통제 정비 필요성을 판단해야 한다.
    
- 목표 :
    - 삭제 조문 탐지
    - 의무 완화 여부 분석
    - 내부 규정 정비 필요성 판단
- 사전 조건 :
    - 기존 법령 DB 최신
    - 조직 내부 규정 해당 조문 반영 중
- 상세 흐름
    
    
    | 단계 | 사용자 행동 | 시스템 동작 |
    | --- | --- | --- |
    | 1 | 입법예고 입력 | 유형 분류 |
    | 2 | Change Analyst | 삭제 조문 탐지 |
    | 3 | Risk Analyst | 의무 완화 분석 |
    | 4 | Control Auditor | 내부 규정 영향 판단 |
    | 5 | Action Strategist | 조치 필요성 제안 |
- 입력 예시
    
    ```yaml
    제18조 제4항 삭제
    ```
    
- 기대 출력
    
    ```
    [변경 요약]
    - 제18조 제4항 삭제
    
    [리스크 분석]
    - 기존 의무 조항 제거
    - 관리 부담 감소
    리스크 등급: Low
    
    [조직 영향]
    - 내부 규정에 해당 조문 인용 중
    - 정비 필요
    
    [Action]
    1. 내부 규정 개정 검토
    2. 불필요 통제 제거 여부 판단
    ```
    
- 성공 기준
    - 삭제 조문 100% 탐지
    - 내부 규정 영향 판단 정확도 90% 이상
